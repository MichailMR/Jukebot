# Jukebot
A discord bot that plays live radio from audio streams.
To use the bot create a bot and generate a token at https://discord.com/developers/ and put this token in key.json at "key".

Prefix: ?

Commands:
- help
- radio / r
- search / s [search query]
- tuner / t
- streamurl [url]
- like WIP
- dislike WIP
